Descripción del Cargo: Desarrollador Fullstack

Buscamos un Desarrollador Fullstack apasionado y motivado para unirse a nuestro equipo. El candidato ideal debe tener experiencia en el desarrollo de aplicaciones web utilizando tecnologías modernas y ser capaz de trabajar tanto en el frontend como en el backend.

Responsabilidades

  • Desarrollar aplicaciones web utilizando Node.js, HTML, CSS y JavaScript.
  • Implementar y gestionar bases de datos utilizando Sequelize y PostgreSQL.
  • Crear y mantener APIs RESTful para la comunicación entre el frontend y el backend.
  • Colaborar con diseñadores y otros desarrolladores para mejorar la experiencia del usuario.
  • Realizar pruebas y depuración de aplicaciones para garantizar su funcionalidad.
  • Participar en revisiones de código y contribuir a la mejora continua del equipo.

Requisitos Técnicos

  • Experiencia sólida en Node.js y su ecosistema.
  • Dominio de HTML, CSS y JavaScript (ECMAScript 5 y 6).
  • Conocimiento en el uso de Bootstrap para el desarrollo de interfaces responsivas.
  • Experiencia en el uso de Sequelize para la gestión de bases de datos PostgreSQL.
  • Familiaridad con control de versiones usando Git, GitHub y GitLab.
  • Conocimiento en el desarrollo de APIs RESTful.

Habilidades Blandas

  • Excelentes habilidades de comunicación y trabajo en equipo.
  • Capacidad para resolver problemas de manera efectiva.
  • Adaptabilidad y disposición para aprender nuevas tecnologías.
  • Gestión del tiempo y organización en el trabajo.
  • Actitud proactiva y enfoque en resultados.

Si cumples con los requisitos mencionados y estás listo para asumir nuevos retos, ¡te invitamos a postularte y unirte a nuestro equipo!
